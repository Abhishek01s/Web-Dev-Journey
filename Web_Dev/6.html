"background-color" property defines the background color for an element

Example: Red background paragraph

"color" property defines the color of text inside element

Example: Olive text

In color related properties color can be specified by name, rgb(a) values, hsl(a) values or hexcode

"font-family" property defines the font to be used for the element

Example: Maple Mono Font

"font-size" property defines the text size for an element

Example: 1.2x larger text

"text-align" property defines the horizontal text alignment for an element:

Example: Center Aligned Paragraph
