If you are using single "p" tag to write multiple lines or/and whitespaces like this (see code), the browser will just ignore them.

"br" tag should be used to tell browser to move to next line

"pre" tag is used to display preformatted text, it will preserve new line and spaces

Example of "pre" tag
             This text contains multiple lines
             and    whitespaces

"hr" or horizontal rule tag is used as thematic break or to seperate contents
