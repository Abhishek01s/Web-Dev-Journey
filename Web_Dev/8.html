CSS is used to styling elements

It can be added to HTML in 3 ways:

Inline: by using "style" attr inside element. Already covered

Internal: by using "style" tag inside "head" tag

External: by using "link" tag inside "head" tag to link external css.

Few CSS properties

"border" property defines a border around a html element

Example: paragraph with borders

"padding" property defines padding (space) between the text and the border

Example: paragraph with borders and 8px padding

"margin" property defines a margin (space) outside the border.

Example: paragraph with borders and 16px margin
